> **Communities of color are 105% more likely to have payday lenders nearby.** The lending industry profits from complexity and desperation. No consumer defense tools exist.

## The Solution

Loan term translator, predatory pattern detector, debt spiral warning, alternative resource finder, CFPB complaint filer.

## Agents

**LoanAnalyzer** — Translate any lending agreement into plain-language true cost (APR, total repayment)
**PredatoryDetector** — Map lender density against demographics, flag predatory patterns
**DebtWarning** — Early warning of debt spiral from financial data
**AlternativeFinder** — Credit union CDFIs, emergency assistance, employer advance programs
**ComplaintFiler** — Generate and file CFPB and state AG complaints
**StudentLoanOptimizer** — Compare IDR plans, find forgiveness eligibility

## Safety

This is financial information, not financial advice. Consult a licensed financial advisor for personalized guidance.

## Data Sources

CFPB complaint database, HMDA data, FDIC/NCUA data, Census demographics, CDFI Fund directory
